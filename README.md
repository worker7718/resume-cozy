# Resume Cozy

A warm, painterly portfolio homepage with a nostalgic studio atmosphere.

## Preview

Open `index.html` directly in a browser, or serve the folder locally:

```sh
npx serve .
```

The current homepage uses a cozy desk/window hero scene, a featured work grid, and a closing contact band.

## Project Structure

- `index.html` - homepage markup
- `styles.css` - site styling and responsive layout
- `assets/` - optimized hero and portfolio imagery
- `images/` - original reference image
